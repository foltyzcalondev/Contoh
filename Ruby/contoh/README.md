# **Contoh v1.0.1**
**Ini adalah contoh library Ruby yang paing simpel dan sederhana :)**

**Library ini ditujukan untuk testing, debugging, dan logging sederhana di terminal.**
> Contoh penggunaan:
```{ruby}
   require 'contoh'

   c = Contoh
   c.contoh()
   # Atau bisa langsung: Contoh::contoh
   c.halo # memunculkan pesan "Halo Dunia!" di terminal.
   c.sapa("John Doe", 30, "Developer") # untuk menyapa dengan identitas yang diberikan dengan field: nama, umur, alamat.
```

## **Fitur utama library:**
- [x] Logging dan testing pesan sederhana
- [x] Dapat menjadi sampel atau contoh library sederhana.
- [x] Dapat memunculkan pesan yang berisi informasi acak di terminal :).

✏️ <mark style="background-color: blue;">Tambahan:</mark> setelah library ini terinstall, maka terdapat perintah baru yang ditambahkan diterminal anda, cukup jalankan:
`$ contoh`
