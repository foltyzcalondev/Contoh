# Contoh
**Ini adalah contoh library Dart yang simpel dan sederhana.**
**Library ini di buat hanya sekedar sebagai contoh atau sampel saja.**
> Contoh penggunaan:

```{dart}
   import "package:contoh/contoh.dart" as contoh;

   void main(List<String> args) {
       contoh.contoh(); # output: Ini adalah contoh library dart :)
       halo(); # output: Halo Dart!
       sapa("Budi", 25, "Jakarta"); # output: Halo!, nama saya Budi, umur saya 25 tahun, dan alamat saya di Jakarta :).
   }
```

## Fitur Utama Library:
* Logging dan debugging sederhana di terminal.
* Dapat dijalankan dengan perintah: `dart pub run contoh`.
* Sangat fleksibel dan minimalis untuk sebagai contoh library Dart.
