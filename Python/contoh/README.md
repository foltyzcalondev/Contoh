Ini adalah contoh library python yang paling simpel, sederhana, dan digunakan sebagai sampel/contoh.
Ditujukan untuk logging, testing, eksperimen, dll.
> Contoh Penggunaan:
```{python}
  from contoh import Contoh
  from contoh import contoh

  c = Contoh()
  c.contoh() # untuk menampilkan pesan sederhana.
  # atau bisa langsung: contoh()
  contoh.sapa() # untuk menyapa anda dengan hangat :)
```
__Catatan: Apabila library ini terinstall, maka terdapat alat baris perintah (CLI) yang akan ditambahkan di sistem terminal anda__, yaitu:
`$ contoh`

## Fitur utama:
* Debugging dan logging sederhana
* Menyapa dengan hangat kepada anda
* Memberi pengetahuan acak mengenai pemrograman dan coding :)
